# Ideas for creating Kentucky Kimchi

## Basic Kimchi recipe
Napa Cabbage
Salt
Garlic
Chili peppers
Fish sauce
(Ginger)

## Ideas for Kentucky Versions
Cabbage - 24 units
Salt - 5 units
Water - 1 liter
Garlic - 1 unit
Dried Jalapeno - 1 unit
Little Bait Fish (cooked) - 1 unit

And let's give it the same nutritional value as 1 cabbage.
But the thirst should be 0
and happiness should be 10
and days fresh should be 21
and days totally rotten should be 730
if possible, the happiness should go up to 20 when it is stale. Otherwise, we will set happiness to 20 from the start. 

In version 2 we should investigate whether we can add ginger as an optional ingredient.
