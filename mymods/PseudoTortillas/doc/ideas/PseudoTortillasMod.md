# PseudoTortillas Mod

We wish to create a mod that is about tortillas.

1. We want to be able to create tortillas
2. We want to be able to make soft tacos with tortillas


Raw Tortilla Recipe:
Flour - 1 unit
Salt - 1 unit
Water - 0.25 liters (water is measured in liters as the default unit)
Butter - 1 stick

Raw tortillas must be cooked before they become regular Vanilla tortillas. They cook as quickly as eggs cook.

Soft Tacos:
Should be like vanilla Project Zomboid tacos, except created with tortillas instead of taco shells
